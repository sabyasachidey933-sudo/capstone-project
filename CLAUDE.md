# Project Rules

## Stack
- HTML
- CSS
- JavaScript
- Node.js

## Coding Conventions

- Use camelCase for variables.
- Write meaningful commit messages.
- Use ES6 syntax.

## Additional Project Rules

1. Always validate user input before submitting forms.
2. Use semantic HTML with labels for accessibility.
3. Review AI-generated code and verify it before committing.
4. Write meaningful Conventional Commit messages.
